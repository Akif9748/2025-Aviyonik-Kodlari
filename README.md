# 2025 Aviyonik Kodları
Pusula Roket Takımı 2025 Aviyonik Kodları Deposu.
UYARI: Bu kodlar 6 eylül 2025 itibari ile güncellenmiştir.
Fakat bu kodların doğrudan kullanımı önerilmez. Bu kodlar alanda kullanıldığı şekilde yüklendiği için
algoritmik olarak test edilip, kendi donanımınıza göre konfigüre edilerek kullanılmalıdır. Yani kodlar ek düzenlemeye uğramadan yüklenmiştir.

Kodlar, uçuşta kullanılanlarla birebir şekilde olup, sadece
MAX3232 ve web arayüzü dahil edilmemiştir.
Bunların sebebi:
- MAX3232 her sene değişiyor, optimizasyon işlemi yapılıp eklenebilir.
- web arayüzü zorunlu değil, fakat alıcı kısmı olarak biz web arayüz kullanıyoruz. Çoğu roket takımı direkt bilgisayara alma konusunda zorlanacağı için eklemedik.


Kodlar, roket verici, görev yükü verici ve okuyucuları ve roket debuglamak için saf veriden oluşmaktadır.
ROKET_VERICI içindeki debug mod açılarak kod daha rahat test edilebilir.
